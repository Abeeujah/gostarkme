# Go Stark Me - Backend Setup 💻

### Steps to Run the Backend 🥳

1. **Navigate to the Contracts Directory 🔍**
   ```bash
   cd gostarkme/contracts
2. **Compile gostarkme backend 🛠️**
   ```bash
   scarb build
3. **Run gostarkme unit test ✅**
   ```bash
   snforce test
4. **Run code formatter 📝**
   ```bash
   scarb fmt
